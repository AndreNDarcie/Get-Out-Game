# Get Out

> A Minimalist Survival Text Game

## Getting Started

Install Parcel using npm
```
npm install -g parcel-bundler
```

```
parcel index.html
```

## Testing

```
npm run test
```

## Deploy to Github Pages

```
npm run deploy
```
